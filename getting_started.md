# Getting started

1) Create a new Python 3 virtual environment

    On Mac or Linux:
    
    ``` python3 -m venv venv ```

    On Windows:
    ```py -m venv venv```

2) Activate Python virtual environment

    On Mac or Linux:
    
    ```source venv/bin/activate ```

    On Windows:
    ```venv/Scripts/activate```

3) install the grabbags in development mode

    ```python -m pip install -e .```

4) Start hacking
